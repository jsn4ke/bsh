- 成立以来净值 (estabRatio)
- 风险等级 (riskLevel)
- 币种 (currType)
- 起售金额 (pfirstAmt)

#### 3.2 APIResponse (Pydantic)
- code: int
- success: bool
- data: dict

  - total: int
  - size: int
  - current: int

**测试点**:

### 6. 年化计算器 (Task 6)
**分支**: `feat/yield-calculator`

#### 6.1 YieldCalculator 类
- `calculate_from_nav(initial_nav, current_nav, days)` - 从净值计算年化
- `calculate_7_day_yield(historical_navs)` - 计算7日年化
- `calculate_3_day_yield(historical_navs)` - 计算3日年化

#### 6.2 计算公式
```
年化收益率 = (当前净值 / 初始净值 - 1) × (365 / 天数) × 100%
```

**测试点**:
- 计算公式正确
- 边界值处理（天数为0、负值等）
- 数值精度正确
- 异常输入处理

---

### 7. Repository 接口设计 (Task 7)
**分支**: `feat/repository-interface`

#### 7.1 BaseRepository (抽象基类)
- 方法：
  - `save(product: ProductModel)` - 保存单个产品
  - `save_batch(products: list[ProductModel])` - 批量保存
  - `find_by_code(code: str) -> ProductModel` - 按代码查找
  - `find_all() -> list[ProductModel]` - 查找所有
  - `update(product: ProductModel)` - 更新产品
  - `delete(code: str)` - 删除产品

#### 7.2 RepositoryFactory
- 根据 配置创建对应的 Repository 实例

**测试点**:
- 接口定义完整
- 工厂模式正确实现
